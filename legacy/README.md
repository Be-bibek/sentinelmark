# SentinelMark v1 (Legacy)

This directory preserves the original SentinelMark v1 codebase.

SentinelMark started as a **cryptographic watermarking and verification system** for
digital assets. The core Rust implementation (`core-rs`) embedded signed watermarks
into data payloads, while the Python verifier (`verify-py`) validated them.

## Evolution

| Version | Focus |
|---------|-------|
| v1 | Cryptographic Watermarking & Verification |
| v2 | Behavior-Aware Continuous Trust Infrastructure Platform |

This evolution story demonstrates how a focused cryptographic tool grew into a
full trust infrastructure SDK suitable for enterprise, fintech, and blockchain
authorization use cases.
